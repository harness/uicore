## Usage Retaining String Input Duration and Supports Expression Value

```jsx live=true
<div id="duration-input-1">
  <DurationInput
    valueInTimeFormat="2d 2h"
    allowVariables={true}
    allowedUnits={['w', 'd', 'h', 'm']}
    placeholder="Enter w/d/h/m or an expression (${...})"
    onChange={time => console.log(time)}
  />
</div>
```
