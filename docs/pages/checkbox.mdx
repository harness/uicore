# Checkbox --- docs/pages/checkbox.mdx

Checkbox component reuses Blueprint's Checkbox and implements the look and feel of [HDL](https://projects.invisionapp.com/d/?origin=v7#/projects/prototypes/17628604).
Checkbox API is exactly the same as [Blueprint's Checkbox](https://blueprintjs.com/docs/#core/components/checkbox).

## Import

```js
import { Checkbox } from '@wings-software/uicore'
```

## Checkboxes

```js live=true
<Checkbox label="Not Selected"/>
<Checkbox label="Selected" checked/>
<Checkbox label="Disabled" disabled/>
<Checkbox label="Disabled and Selected" disabled checked />
```
