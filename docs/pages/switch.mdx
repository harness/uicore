# Switch --- docs/pages/switch.mdx

Switch component reuses Blueprint's Switch and implements the look and feel of [HDL](https://projects.invisionapp.com/d/?origin=v7#/projects/prototypes/17628604).
Switch API is exactly the same as [Blueprint's Switch](https://blueprintjs.com/docs/#core/components/switch).

## Import

```js
import { Switch, SwitchIcon } from '@wings-software/uikit'
```

## Switches

```js live=true
<Switch label="Not Selected" />
<Switch label="Selected" checked/>
<Switch label="Disabled" disabled/>
<Switch label="Disabled and Selected" disabled checked />
```


## Switch Icon

```
export interface SwitchIconProps extends IControlProps {
  icon: IconName
  iconChecked: IconName
  iconProps: IconProps
}
```

```js live=true
<b>Default View - Without Label</b>
<SwitchIcon icon="play" iconChecked="command-rollback" />
<b>Default Checked</b>
<SwitchIcon icon="play" iconChecked="command-rollback" defaultChecked labelElement={<div>Rollback</div>} />
<b>Label</b>
<SwitchIcon icon="play" iconChecked="command-rollback" label="Rollback" onChange={(e) => console.dir(e.target.checked)} />
<b>Disabled</b>
<SwitchIcon icon="play" iconChecked="command-rollback" label="Rollback" disabled onChange={(e) => console.dir(e.target.checked)} />
<b>Disabled with checked</b>
<SwitchIcon icon="play" iconChecked="command-rollback" checked label="Rollback" disabled onChange={(e) => console.dir(e.target.checked)} />
```
