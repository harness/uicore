# Horizontal --- docs/pages/layout-horizontal.mdx

`<Layout.Horizontal/>` lays out children in a horiztonally spaced layout.

```js live=true
<Layout.Horizontal margin="large">
  <div style={{ height: '100px', width: '200px', background: 'var(--grey-300)' }}></div>
  <div style={{ height: '100px', width: '300px', background: 'var(--green-500)' }}></div>
  <div style={{ height: '100px', background: 'var(--blue-500)', flexGrow: 1 }}></div>
</Layout.Horizontal>
```
