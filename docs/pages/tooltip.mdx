## Tooltip Triggered By Click

```js live=true
<Button
  text="Delete"
  intent="danger"
  tooltip={
    <Container width="350px" padding="medium">
      <Heading level={2} font={{ weight: 'semi-bold' }} margin={{ bottom: 'small' }}>
        Confirm deletion
      </Heading>
      <Text margin={{ bottom: 'medium' }}>
        Are you sure you want to delete these items? You won't be able to recover them.
      </Text>
      <Container flex>
        <span />
        <Layout.Horizontal spacing="small">
          <Button text="Cancel" className="bp4-popover-dismiss" />{' '}
          {/* You should always use Blueprint Classes.POPOVER_DISMISS */}
          <Button intent="danger" text="Confirm" />
        </Layout.Horizontal>
      </Container>
    </Container>
  }
  tooltipProps={{
    interactionKind: 'click'
  }}
/>
```
