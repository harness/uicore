## Font Family

[Nunito](https://fonts.google.com/specimen/Nunito?selection.family=Nunito) is Harness default text font. This font is enforced across the whole document so when you build your component, you don't need to specify it.

## Font Sizes

UIKit defines four font sizing CSS variables.

```css
--font-size-small: 12px; /* H4, small text */
--font-size-normal: 14px; /* H3, normal text */
--font-size-medium: 18px; /* H2 */
--font-size-large: 29px; /* H1 */
```

These CSS variables are used internally inside UIKit. When you build a component, or use a UIKit component, you should always set these sizes with the `font` prop. For example:

```js live=true
<Text font={{ size: 'large', align: 'center' }}>The quick brown fox jumps over the lazy dog</Text>
```
