# DateInput --- docs/pages/dateinput.mdx

DateInput use cases

- Show date and date time and return response in milliseconds in strings
- Use Calender for date and date time selection
- Uses same props as TextInput
- Support for custom variables like current() & ${variable} workflow variables
  - Plus/Minus Weeks/Days/Hours/Minutes/Seconds similar to DurationInput
  - Returns the response as current() ${variable} +- duration
- Show valid date input

## Import

```js
import { DateInput } from '@wings-software/uicore'
```

## Show Date

```js live=true
<DateInput />
```

## Show Time

```js live=true
<DateInput timePrecision="minute" />
```

## Show Date with value

```js live=true
<DateInput value="1579773229467" />
```

## Show Time with value

```js live=true
<DateInput timePrecision="minute" value="1579773229467" />
```

## Show Date with Custom Variables

```js live=true
<DateInput allowVariables={true} onChange={(value, error) => console.log(value, error)} />
```

## Show Time with Custom Variables

```js live=true
<DateInput timePrecision="minute" allowVariables={true} onChange={(value, error) => console.log(value, error)} />
```

## Show Date with Custom Variables with value

```js live=true
<DateInput allowVariables={true} value="current() + 7200000" onChange={(value, error) => console.log(value, error)} />
```

## Show Time with Custom Variables with value

```js live=true
<DateInput
  timePrecision="minute"
  allowVariables={true}
  value="${App.TIME_LOCAL} - 777600000"
  onChange={(value, error) => console.log(value, error)}
/>
```

## Show Date with Custom Variables with error value

```js live=true
<DateInput allowVariables={true} value="current() + 2h 10m + 2" />
```

## Show Time with Custom Variables with error value

```js live=true
<DateInput timePrecision="minute" allowVariables={true} value="current() + w X" />
```
