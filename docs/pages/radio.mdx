# Radio --- docs/pages/radio.mdx

Radio component reuses Blueprint's Radio and implements the look and feel of [HDL](https://projects.invisionapp.com/d/?origin=v7#/projects/prototypes/17628604).
Radio API is exactly the same as [Blueprint's Radio](https://blueprintjs.com/docs/#core/components/radio).

## Import

```js
import { Radio, RadioGroup } from '@wings-software/uicore'
```

## Single Radio

```js live=true
<Radio label="Item 1" value="one" />
```

## Multiple Radios

```js live=true
<RadioGroup label="Section Headline" selectedValue={'other'}>
  <Radio label="Disabled" value="one" disabled />
  <Radio label="Not Selected" value="three" />
  <Radio label="Selected" value="two" defaultChecked={true} />
  <Radio label="Disabled and Selected" value="four" defaultChecked={true} disabled />
</RadioGroup>
```
