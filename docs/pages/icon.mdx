# Icon --- docs/pages/icon.mdx

All icons in UICore are in svg format. You can use the icons as a separate React component or using them in [Button](/button).

Note that `<Icon/>` component supports multiple colors in svg file, but when combining with `<Button/>` component, fill and stoke colors in svg files are reset to the text color of the button.

To use a Harness icon, first import `Icon` from UICore.

```js
import { Icon } from '@wings-software/uicore'
```

Then render icon as a React component.

```js live=true
<Layout.Horizontal spacing="small" id="color-icons">
  <Icon name="service-gotlab" size={64} />
  <Icon name="service-datadog" size={64} />
  <Icon name="service-github" size={64} />
  <Icon name="service-gcp" size={64} />
  <Icon name="service-jenkins" size={64} />
  <Icon name="service-slack" size={64} />
  <Icon name="spinner" size={64} color="blue500" />
  <Icon name="gear" size={64} color="green500" />
  <Icon name="hourglass" size={64} color="green500" />
  <Icon name="pending" size={64} color="grey300" />
</Layout.Horizontal>
```

You can pass styled props to Icon as well.

```js live=true
<Icon id="icon-styled-props" name="heart" size={64} color="blue500" padding="small" border={{ color: 'blue400' }} />
```

## Harness Logos

UICore currently has two Harness logos which are not suitable to be rendered by `<Icon />` due to the fact that the logos are not square. To render these logos, you can use their SVG components directly from `HarnessIcons['harness-logo-black']` and `HarnessIcons['harness-logo-white']`.

```js render=true
<div style={{ background: 'var(--grey-100)', padding: '20px', borderRadius: '5px' }}>
  <Layout.Horizontal spacing="small">
    {[HarnessIcons['harness-logo-black'], HarnessIcons['harness-logo-white']].map((HarnessLogo, index) => (
      <HarnessLogo height={48} key={index} />
    ))}
  </Layout.Horizontal>
</div>
```

## How to add an icon?

To add an icon into UICore, copy the icon into UICore's `./src/icons` folder then run

```
yarn ui:icons
```

## Harness Icons

```js render=true
<div
  id="harness-icons"
  style={{
    padding: '20px',
    display: 'grid',
    background: 'var(--grey-100)',
    gridGap: '20px',
    gridTemplateColumns: 'repeat(auto-fit, minmax(200px, 1fr))',
    borderRadius: '5px'
  }}>
  {Object.keys(HarnessIcons).map(name => (
    <button
      style={{
        all: 'unset',
        justifySelf: 'center',
        alignSelf: 'center',
        display: 'inline-block',
        cursor: 'pointer'
      }}
      onClick={() => Utils.copy(name)}>
      <Popover
        key={name}
        interactionKind="hover"
        content={<Text padding="small">{name} (click to copy)</Text>}
        usePortal={false}>
        <Layout.Vertical
          spacing="small"
          style={{
            justifyContent: 'center',
            alignItems: 'center'
          }}>
          <Icon name={name} size={24} padding="small" border={{ color: 'grey300' }} />
          <Text font="small">{name}</Text>
        </Layout.Vertical>
      </Popover>
    </button>
  ))}
</div>
```
