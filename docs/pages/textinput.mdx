# TextInput --- docs/pages/textinput.mdx

## Import

```js
import { TextInput } from '@wings-software/uicore'
```

## Idle state

```js live=true
<TextInput defaultValue="" placeholder="Placeholder text" />
```

## Error state

```js live=true
<TextInput defaultValue="Now in error state" intent="danger" errorText="Not a valid data. Please try again." />
```

## Error state (in popover)

```js live=true
<TextInput
  defaultValue="Show Error icon"
  errorText="Not a valid data. Please try again."
  intent="danger"
  errorInPopover
/>
```

## Success state

```js live=true
<TextInput placeholder="Placeholder text" intent="success" />
```

## Disabled state

```js live=true
<TextInput placeholder="Placeholder text" disabled />
```

## Icons

```js live=true
<TextInput
  placeholder="Search"
  leftIcon="search"
  rightElement={<Button minimal icon="arrow-right" intent="primary" margin="none" />}
```
