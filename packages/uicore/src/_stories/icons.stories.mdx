All icons in UICore are in svg format. You can use the icons as a separate React component or using them in [Button](/button).

Note that `<Icon/>` component supports multiple colors in svg file, but when combining with `<Button/>` component, fill and stoke colors in svg files are reset to the text color of the button.

To use a Harness icon, first import `Icon` from UICore.

```js
import { Icon } from '@harness/icons'
```

Then render icon as a React component.

```js
<Layout.Horizontal spacing="small" id="color-icons">
  <Icon name="gear" size={64} color="green500" />
  <Icon name="hourglass" size={64} color="green500" />
  <Icon name="pending" size={64} color="grey300" />
</Layout.Horizontal>
```

<Layout.Horizontal spacing="small" id="color-icons">
  <Icon name="gear" size={64} color="green500" />
  <Icon name="hourglass" size={64} color="green500" />
  <Icon name="pending" size={64} color="grey300" />
</Layout.Horizontal>

You can pass styled props to Icon as well.

```js
<Icon id="icon-styled-props" name="heart" size={64} color="blue500" padding="small" border={{ color: 'blue400' }} />
```

## How to add an icon?

To add an icon into UICore, copy the icon into UICore's `./src/icons` folder then run

```
yarn ui:icons
```
